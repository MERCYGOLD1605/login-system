# 🔐 Login System

A simple Flask-based web application that allows users to register, login, and logout.

---

## 🚀 Features
- User Registration  
- User Login  
- Session Management  
- Logout functionality  

---

## 🛠 Tech Stack
- Python  
- Flask  
- JSON (for storage)  

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
